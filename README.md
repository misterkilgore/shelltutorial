# Lezioni di Shell Scripting

Questa repository contiene una serie di lezioni pratiche dedicate allo **shell scripting** su sistemi Unix/Linux. L’obiettivo è fornire esempi semplici e chiari per apprendere i concetti fondamentali e avanzati della programmazione in shell.

---

## Contenuti

- `arguments.sh`  
  Esempi e spiegazioni sull’uso degli **argomenti passati agli script**.

- `execution_error.sh`  
  Gestione degli **errori di esecuzione** e controllo dello stato di uscita degli script.

- `fileread.sh`  
  Lettura e manipolazione di file di testo all’interno di uno script.

- `flow.sh`  
  Strutture di controllo del flusso: condizioni, cicli e branch.

- `functions.sh`  
  Definizione e utilizzo di **funzioni** in shell scripting per modularità e riuso.

- `ips.txt`  
  File di esempio contenente indirizzi IP utilizzato nelle lezioni per esercitazioni pratiche.

- `template.sh`  
  Modello elegante di script shell per iniziare rapidamente nuovi progetti.


## Scopo e destinatari

Questi esempi sono pensati per l'utilizzo nel mondo dell'automazione IT.
